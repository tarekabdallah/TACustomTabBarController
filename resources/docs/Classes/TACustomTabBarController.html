TACustomTabBarController

open class TACustomTabBarController: UIViewController

TACustomTabBarController can be used insead of the default tab bar contoller. It has some additional features such as text position with respect to the icon, rach tabBar item Background color, text color and text color. seperator color and tab bar height. This custom tab bar also enables adding a starting view controller which is not in the tab bar and allows jumping to it at anytime while handling selection.
